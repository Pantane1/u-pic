# U-Pic

**[Live Demo](https://u-pic.gt.tc/)**

## Overview

U-Pic is a simple web application that allows users to upload, view, and download images directly from a web interface. All uploaded images are saved in the project’s public directory, making them easily accessible through the browser.

The project is ideal for managing and sharing images quickly without the need for complex backend setups.

## Features

* Upload images to the server (`public/` folder).
* View all saved images in a gallery view.
* Download any image directly from the gallery.
* Clean, intuitive, and user-friendly interface.
* Lightweight and simple deployment process.

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** PHP
* **Hosting:** InfinityFree

## How to Use

1. Open the [live link](https://u-pic.gt.tc/)
2. Use the upload button to select images from your computer.
3. Uploaded images will appear in the gallery section.
4. Click the “Download” button below any image to save it locally.

## Deployment

1. Upload all project files to your hosting server (e.g., InfinityFree).
2. Ensure the `public/` folder exists and has proper write permissions.
3. Access the live site through your domain link to start uploading and viewing images.

## Notes

* Ensure uploaded images meet the server’s file size limits.
* Supported image formats: JPG, PNG, GIF (can be extended if needed).
